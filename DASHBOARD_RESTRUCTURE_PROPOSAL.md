## ğŸ¨ Visual Design Principles

### Page Layouts Should Follow:

**1. Progressive Disclosure:**
- Start with high-level overview
- Drill down into details
- Always show context breadcrumbs

**2. Glass Morphism Hierarchy:**
```
Page Background: #FAFAFA (warm ivory)
    â†“
Primary Cards: bg-white/50 backdrop-blur-[16px]
    â†“
Nested Cards: bg-white/80 (slightly more opaque)
    â†“
Emphasized Elements: bg-white (full white)
```

**3. Information Density:**
- Home: High-level, scannable
- Connect: Medium, action-focused
- Soul Signature: Deep, exploratory
- Revelation Controls: Detailed, configurable
- Twin Chat: Conversational, interactive

**4. Color Accents:**
- Primary: Stone-900 (dark anchors)
- Secondary: Stone-600 (muted text)
- Highlights: Use subtle opacity changes, not bright colors
- Status: Stone palette (not green/red)
