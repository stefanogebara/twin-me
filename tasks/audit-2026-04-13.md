## HIGH

**H1 — CORS allows all browser extensions**
`api/server.js` ~line 150 whitelists all `chrome-extension://` and `moz-extension://` origins.
Any installed extension can make credentialed API requests. JWT is the backstop, but stolen token + malicious extension = full access.
Fix: Remove blanket extension allow or gate behind specific extension IDs.

**H2 — DEV_USER_ID hardcoded UUID fallback in nangoService.js**
`api/services/nangoService.js:46,74` falls back to hardcoded UUID `167c27b5-...` if env var absent.
If `NODE_ENV` is undefined on a staging deploy, real user OAuth credentials exposed.
Fix: Remove UUID fallback, assert env var is set in dev.

## MEDIUM

**M1 — 4 platform tokens expired (github, discord, reddit, linkedin)**
`/api/connectors/status` returns `token_expired` for 4 platforms.
Connect page renders expired UI. Not a code bug — needs manual re-auth by the user.

**M2 — /brain unreachable from any navigation**
Route `/brain` works but is not linked in `CollapsibleSidebar` or `BottomNav`.
Users can only reach it via direct URL entry.

**M3 — /twin/insights 13s cold latency (in-process cache only)**
`insightsCache` Map resets on every Vercel cold start.
Production users hit 13s on function instance spin-up.
Fix: Persist cache in Vercel KV or Supabase with TTL.
