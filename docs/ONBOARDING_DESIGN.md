# Soul Signature Onboarding - Progressive Revelation Design

## Design Philosophy

The onboarding should feel like **a journey of self-discovery**, not a data collection form. Each step reveals something about the user while building anticipation for their full Soul Signature.

---

## Visual Theme: "Constellation of Self"

### Core Metaphor
Your Soul Signature is like a constellation - individual stars (data points) that together form a meaningful pattern unique to you.

### Color Palette
```
Primary Gradient: #8B5CF6 (violet) â†’ #EC4899 (pink) â†’ #F59E0B (amber)
Background: Dark (#0c0a09) with subtle star particles
Accent: Soft glow effects, aurora-like gradients
Text: Warm white (#fafaf9) with muted secondary (#a1a1aa)
```

### Animation Style
- Smooth, organic transitions (300-500ms ease-out)
- Particle effects for "discovery" moments
- Gentle pulse animations for interactive elements
- Constellation lines drawing between connected insights
